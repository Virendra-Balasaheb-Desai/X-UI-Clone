# X (Twitter) UI Clone
This X clone is `UI` clone only and single page design.
This project is part of Sigma Web Development by CodeWithHarry.
It uses `Tailwind CSS`.

## Project Setup
### Install packages
```
npm install
```
### Run project
```
npm run dev
```
## To make changes in project

### To edit CSS (Tailwind CSS)
```
npm run build
```
Start editing CSS.
